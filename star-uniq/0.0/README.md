# Star Uniq

Filter out repeated triangles.

## Requirements

- C compiler
- POSIX make
- pkg-config
- [RSys](https://gitlab.com/vaplv/rsys/)

## Installation

Edit config.mk as needed, then run:

    make clean install

## Release notes

### Version 0.0

Save triangles once and structure them in an indexed mesh
